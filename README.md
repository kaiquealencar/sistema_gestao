# Sistema Vitrine – Gestão de Clientes
## Projeto em desenvolvimento

Sistema web desenvolvido em **Django** com o objetivo de gerenciar clientes e gerar crachás de identificação.  
Este projeto faz parte do meu portfólio e está sendo desenvolvido de forma incremental, seguindo boas práticas de versionamento e organização de código.

## Funcionalidades atuais

- CRUD completo de clientes (Criar, Ler, Atualizar, Excluir)  
- Mensagens de feedback para ações realizadas (success, error)  


## Funcionalidades planejadas
- CRUD completo de serviços
- Controle de permissões por tipo de usuário (Admin, Gerente, Funcionário)  
- Relatórios de clientes e crachás emitidos  
- Integração com QR Code  
- Deploy em servidor web

**Tecnologias**
- Python 3.14.0
- Django 6.0.1
- SQLite
- HTML / CSS / Bootstrap 5  
- JavaScript (validações e interações básicas)
